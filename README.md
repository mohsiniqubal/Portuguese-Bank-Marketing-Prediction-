Portuguese Bank Marketing Prediction using Machine Learning
________________________________________
Project Overview
This project predicts whether a customer will subscribe to a term deposit based on data collected from direct marketing campaigns conducted by a Portuguese banking institution.
Using historical customer information and campaign details, multiple machine learning classification models are trained and evaluated to help improve marketing efficiency and customer targeting.
The project covers the complete machine learning workflow, from data preprocessing to model evaluation and prediction.
